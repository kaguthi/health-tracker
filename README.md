# Welcome to HealthTracker 👋

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start | npm start
   ```

## prebuild

```bash
   npx expo prebuild
```

## Running on android

```bash
   npm run android | npx expo run:android
```

## Getting your ipAddress

   ``` bash
   ipconfig
   ```
